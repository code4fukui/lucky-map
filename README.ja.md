# ラッキーマップ

ランダムな地点の情報を表示する地図アプリです。

## デモ
https://code4fukui.github.io/lucky-map/

## 機能
- ランダムな地点の位置情報を地図上に表示します
- 地点の市区町村名を表示し、リンクから詳細情報を確認できます

## データ・API
- 本アプリケーションは、[localgovjp-utf8.csv](https://code4fukui.github.io/localgovjp/localgovjp-utf8.csv)のデータを使用しています。このデータセットには、日本の地方公共団体の情報が含まれています。
